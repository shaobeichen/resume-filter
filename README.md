# resume-filter

> 根据关键字进行简历筛选，提高效率

## 使用

1. 将 pdf 简历放进 files 文件夹

2. 进入项目执行命令行，在 windows 环境可以双击 "一键执行.bat"

```shell
npm i
npm run dev
```
